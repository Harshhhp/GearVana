GearVana is a full-stack car marketplace application that allows users to browse, filter, and explore cars from various brands. It features a modern UI, detailed car information, pricing, EMI calculator, and test drive booking functionality.  

---

## âœ¨ Features

- Browse cars from multiple brands: Hyundai, BMW, Ford, Mahindra, Honda, Tesla, Tata, etc.
- Detailed car information: make, model, year, mileage, fuel type, body type, and price
- Responsive UI built with **Next**, **Tailwind CSS**, and **ShadCN UI**
- Book test drives directly from the app
- EMI calculator for car financing
- Admin dashboard to manage car listings
